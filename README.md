# Kanigiri Cafe

yeah, it's a restaurant website. shocking, i know.

## what's inside

- **HTML** - fancy structure nobody reads
- **CSS** - makes it look less terrible
- **JavaScript** - because why have a static site like it's 2005

## features

- responsive design (works on your phone, tablet, whatever)
- menu that actually updates without crying
- shopping cart that won't lose your order
- checkout form that validates itself
- order history stored locally (so you can torture yourself looking at what you spent)

## the actual stuff

`script.js` - handles all the interactive nonsense
`orders-store.js` - keeps track of orders in localStorage because servers are overrated
`styles.css` - attempts to make things pretty
`index.html` - the main event

## how to use it

open it in a browser. order some biryani. that's it.
