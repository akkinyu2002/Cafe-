# Charminar Brew House

A full cafe-bar-restaurant website for Hyderabad with menu browsing, cart management, and order placement.

## Features
- Premium single-page design with animated hero
- Category filters and search-based menu discovery
- Sorting by popularity, price, and rating
- Quantity selection and cart drawer
- Checkout flow with delivery/pickup/dine-in options
- Delivery zone based fee and ETA
- Table booking form
- Cart persistence via localStorage
- Keyboard shortcuts (`Ctrl/Cmd + K` search, `Ctrl/Cmd + B` cart)
- Responsive design for desktop and mobile

## Local Run
Open `index.html` in any modern browser.

## Tech
- HTML
- CSS
- Vanilla JavaScript
